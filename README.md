# Lm-Code

Инструмент, позволяющий ИИ-чатботу на [arena.ai](https://arena.ai) выполнять реальные операции на локальной машине: работа с файлами, запуск команд, поиск и редактирование кода.

## Структура

Проект состоит из трёх компонентов:

- **lm-cli** - терминальный клиент. Принимает пользовательский ввод, отправляет его в чат через API, получает ответ ИИ, выполняет команды из него и возвращает результат обратно в чат. Цикл повторяется автоматически, пока в ответе есть команды.
- **lm-api** - локальный HTTP-сервер (порт 11856). Парсит и выполняет команды (READFILE, WRITEFILE, RUN, EDIT и др.), обслуживает очереди сообщений между CLI и расширением.
- **lm-addition** - Chrome-расширение. Работает на странице arena.ai: отслеживает ответы ИИ через MutationObserver, находит в них команды, отправляет на lm-api для выполнения, инжектит текст в поле ввода чата.

## Как работает связь

```
Пользователь (lm-cli)
    |
    | POST /queue_inject
    v
lm-api (сервер)
    |
    | GET /pending_inject (polling)
    v
lm-addition (расширение) --> вставляет текст в чат arena.ai
                                        |
                                   ИИ отвечает
                                        |
                              расширение читает ответ
                                        |
                         POST /ai_response (текст + модель)
                                        |
                                        v
                                    lm-api (сервер)
                                        |
                              GET /pending_response
                                        |
                                        v
                                Пользователь (lm-cli)
```

Если в ответе ИИ есть команды - расширение отправляет их на `POST /execute`, lm-api выполняет и возвращает результат. CLI также может выполнять команды напрямую через тот же эндпоинт.

## CLI-команды

| Команда | Описание |
|---------|----------|
| `/ai/send_protocol` | Отправить ИИ описание доступных API-команд |
| `/ai/send_status` | Отправить ИИ информацию о системе (ОС, CPU, RAM, GPU, время) |
| `/ai/new_chat` | Создать новый чат на arena.ai |
| `/ai/model/swap <запрос>` | Найти и выбрать модель по названию (интерактивно) |
| `/checkrain/on` | Включить режим checkrain, он добаляет **в конце** к любому запросу рандомный unicode символ, для обхода капчи |
| `/checkrain/off` | Выключить режим checkrain |
| `/clear` | Очистить консоль |
| `/help` | Список команд |

## API-команды ИИ

`FILELIST`, `READFILE`, `READLINES`, `WRITEFILE`, `APPENDFILE`, `DELETEFILE`, `RUN`, `SEARCH`, `EDITLINES`, `EDIT`, `QUESTIONS`
